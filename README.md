# 基于SpringBoot+Layui的社区物业管理系统

## 介绍
基于SpringBoot+Layui的社区物业管理系统开发

## 技术栈

- 数据库：MySQL
- 后端技术：SpringBoot，MyBatisPlus
- 数据连接池：Druid
- 前端技术：Layui，Ajax，Json等
- Web容器：Apache Tomcat 8.5.64
- 项目管理工具：Maven
- 版本控制工具：Git
- 开发工具：IDEA
